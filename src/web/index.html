duplichecker$ help

Waehle eine Quelle und starte die Suche:

duplichecker$ load --from csv
duplichecker$ load --from spotify

checking...

> spotify credentials einrichten / aendern

Hole dir Client ID + Secret unter developer.spotify.com/dashboard. Wichtig: bei Redirect URIs http://127.0.0.1:8888/callback eintragen (NICHT localhost — das lehnt Spotify ab).

Logge dich mit deinem Spotify-Account ein, um deine eigenen Playlists zu sehen. Es oeffnet sich der Browser zum Anmelden.

duplichecker$ my-playlists

    Klicke auf eine Playlist, um sie zu pruefen.

    Die Playlist muss oeffentlich sein.
